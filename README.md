# MyGpt

1. To create virtual env:-
   -> pip install virtualenv
   -> virtualenv myenv
   -> myenv\Scripts\activate
   (OR)
   open cmd through vscode
   -> run python -m venv myenv
   -> myenv\Scripts\activate
3. cd chatapp
4. run pip install -r requirements.txt
5. cd..
6. run python manage.py runserver
